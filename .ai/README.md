# AI Guidance

This folder contains provider-neutral guidance for AI assistants working on this career documentation repository.

## How to use these files

- Read `rules/*.md` before editing repository content or changing git state.
- Treat `settings.json` as shared project guidance, not as a vendor-specific runtime config.
- Keep local or personal tool permissions out of the repository.
- Prefer small, explicit documentation edits over broad rewrites.

## Project shape

This repository is a Markdown-based career portfolio. Most changes should update existing `.md` files, add project or experience documents, and keep `README.md` navigation accurate.
